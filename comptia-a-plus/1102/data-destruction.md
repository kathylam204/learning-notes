# Data Destruction
## Physical Destruction Methods (list)
- Shredder
- drill/hammer
- Degaussing
- Incineration
## Degaussing
- removes the magnetic field of a disk
- destroys both the data drive and the electronics
## Low-Level Disk Formatting
- provided by the factory
## Standard Drive Formatting Options
- quick format
- regular format
## Quick Format
- sets up the file system and installs a boot sector
- clears the master file table, but not the data
- Recoverable
## Regular Format
- overwrites every drive sector with zeros
- default for Windows
- Non-recoverable
## "sdelete" (command)
- file level overwriting
