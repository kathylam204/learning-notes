# Remote Access
## RDP (Microsoft Remote Desktop Protocol)
- clients for MacOS, Linux, etc.
- remotely connect to a Windows device
## Virtual Network Computing (acronym)
- VNC
## Remote Frame Buffer Protocol (acronym)
- RFB
## VNC
- uses RFB protocol
- clients for many operating systems
## Client-to-Site VPN
- on demand access from a remote device
- software connects to a VPN concentrator
- some software can be configured as always-on
## Secure Shell (SSH)
- tcp/22
- encrypted console communication
- supports public/private key pair authentication
- certain accounts, such as root, should be disabled in SSH
- limit access to SSH by IP address
- using a local firewall or a network filter
## Remote Monitoring and Management (acronym)
- RMM
## Managed Service Providers (acronym)
- MSP
## RMM
- manage a system from a remote location
- often used by MSP's
- RMM access should be limited
- perform ongoing audits
## Microsoft Remote Assistance (acronym)
- MSRA
## MSRA
- provides access to a remote user's desktop
- access on demand
- no firewall configurations or port forwarding required
- replaced with Quick Assist in Windows 10 and Windows 11
