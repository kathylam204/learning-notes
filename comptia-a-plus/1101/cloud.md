# Cloud
## Cloud Computing
- extends the management of applications and services
- deploy an infrastructure in minutes
- international scope
- Cost-effective
## Private Cloud
- your own personalized local data center
## Public Cloud
- available to everyone over the internet
## Hybrid Cloud
- a mix of public cloud and private cloud
## Community Cloud
- several organizations share the same resources
## Infrastructure as a Service (SaaS)
- sometimes called HaaS
- outsource your equipment
- buying time on someone else's hardware
- you're still responsible for management and security
## Software as a Service (SaaS)
- on-demand software
- centralized management of data and applications
- a complete application offering
## Platform as a Service (PaaS)
- the middleground option (between IaaS and SaaS)
- no servers, no software, no maintenance team, no HVAC
- someone else handles the platform, you handle the development
- you don't have direct control over data, people, or infrastructure
- may be module based
## Internal Cloud
- no resources are shared
- build your own cloud
- only up-front costs
## External Cloud
- share resources on a public cloud
- underlying infrastructure is owned by a third-party
- costs may be metered or up-front
## Metered Cloud Services
- you pay for what you use
## Rapid Elasticity
- scale up and down as needed
- seamless to the end user
## Desktop as a Service (DaaS)
- entire Windows desktop runs on the cloud
- applications run on a virtual desktop infrastructure
- big network requirements
