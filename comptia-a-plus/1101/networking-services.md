# Networking Services 2
## File Server
- provides centralized storage of files
- SMB is the usual protocol on Windows devices
- AFP (Apple Filing Protocol) is the usual protocol on Apple devices
- the front-end hides the protocols
## Print Server
- connects a printer to the network
- may be software, or may be built into the printer
- uses standard printing protocols, such as SMB, IPP (Internet Printing Protocol), and LDP (Line Printer Daemon)
## Syslog Server
- standard for message logging
- consolidates logs to a central database (integrated into the SIEM, or the Security Information and Event Manager)
## Web Server
- responds to browser requests
- uses HTTP or HTTPS
- pages are built using HTML or HTML5
- web pages are stored on the server, and downloaded in the browser
## Authentication Server
- centralized management for login authentication
- usually a set of redundant servers
## Spam Gateway
- stops spam before it reaches the user
- categorizes as spam or rejects it
## Proxy Server
- acts as an intermediary between the user and the internet
- processes requests for the user
