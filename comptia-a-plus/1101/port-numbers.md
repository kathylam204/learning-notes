# Port Numbers
## FTP (File Transfer Protocol)
- tcp/20 (active data mode), tcp/21 (control)
## SSH (Secure Shell)
- tcp/22
## Telnet (Telecommunication Network)
- tcp/23
## SMTP (Simple Mail Transfer Protocol)
- tcp/25
## DNS (Domain Name System)
- udp/53
## DHCP (Dynamic Host Configuration Protocol)
- udp/67, udp/68
## HTTP (Hypertext Transfer Protocol)
- tcp/80
## HTTPS (Hypertext Transfer Protocol Secure)
- tcp/443
## POP3 (Post Office Protocol version 3)
- tcp/110
## IMAP4 (Internet Message Access Protocol, version 4)
- tcp/143
## SMB (Server Message Block) & CIFS (Common Internet File System)
- tcp/445
## SNMP (Simple Network Management Protocol)
- udp/161 (queries), udp/162 (traps)
## LDAP (Lightweight Directory Access Protocol)
- tcp/389
## RDP (Remote Desktop Protocol)
- tcp/3389
## NBNAME (NetBIOS Name Service)
- udp/137
## NBSESSION (NetBIOS Session Service)
- udp/139
## Non-Ephemeral Ports
- permanent port numbers
- ports 0 through 1023
## Ephemeral Ports
- temporary port numbers
- determined in real-time by the client
- ports 1024 through 65535
